Synopsis

I worked with a designer and the guys at Potato Potato to create a portfolio site that reflects their brand and highlights the great work they're doing. I learned one huge lesson on this project — read the specs before fully developing a solution! One of the features of the project is the dynamic change of a DOM element on scroll (in this case, their About section). I found a cool CSS property called clip-path that modifies the DOM exactly the way I wanted. After spending a lot of time going this route, I eventually realized clip-path isn't suppported by most browsers and had to completely change directions.
